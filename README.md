# 🧬 Lung and Colon Cancer Classification Using Convolutional Neural Network (CNN)

## 📖 Overview

This project implements a Convolutional Neural Network (CNN) model to classify histopathological images of lung and colon cancer. The model is trained using medical image data to distinguish between cancerous and non-cancerous tissue types.

## 🎯 Objectives

* Develop a CNN-based classification model for histopathological images.
* Analyze the performance of CNN in identifying lung and colon cancer.
* Evaluate model performance using classification metrics.

## 📊 Dataset

This project uses the LC25000 Histopathological Image Dataset.

🔗 Dataset Source:

https://www.kaggle.com/datasets/andrewmvd/lung-and-colon-cancer-histopathological-images

> ⚠️ The dataset is not included in this repository due to its large size. Please download it directly from the source above before running the notebook.

## 🚀 Installation

Clone this repository:

```bash
git clone https://github.com/kknkrnwn/lung-colon-cancer-cnn.git
cd lung-colon-cancer-cnn
```

Install dependencies:

```bash
pip install -r requirements.txt
```

## ▶️ Running the Project

Open Jupyter Notebook:

```bash
jupyter notebook
```

Then execute:

```text
notebooks/lung-colon-cancer-cnn-98-6.ipynb
```
